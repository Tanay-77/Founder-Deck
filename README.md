# FounderDeck

FounderDeck is a Chrome extension that allows you to effortlessly save and organize founder profiles from LinkedIn and X (Twitter) with a single click. View and manage all your saved connections in a beautiful, centralized dashboard.

## Features

- **One-Click Save:** Instantly save profiles directly from LinkedIn and X.
- **Floating Widget:** A convenient, non-intrusive widget injected directly into supported profile pages.
- **Beautiful Dashboard:** A sleek, modern dashboard to browse, search, and manage your saved founders.
- **Data Export & Import:** Easily back up your data by exporting to JSON and import it back whenever you need.
- **Smart Extraction:** Automatically extracts the founder's name, username, bio, and profile picture without any manual entry.

## Installation (Developer Mode)

1. Clone or download this repository to your local machine.
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer mode** using the toggle in the top right corner.
4. Click the **Load unpacked** button and select the folder containing these extension files.
5. The FounderDeck extension is now installed and ready to use!

## How to Use

1. **Pin the Extension:** Pin the FounderDeck icon to your Chrome toolbar for quick access to your dashboard.
2. **Save a Profile:** Visit any user profile on LinkedIn or X (Twitter). You will see the FounderDeck tab appear on the right side of your screen. Click it, then click **Save Founder**.
3. **View Dashboard:** Click the FounderDeck extension icon in your toolbar (or the "Open Dashboard" button in the widget) to view, search, and filter your saved profiles.

## Technologies Used

- Vanilla JavaScript
- HTML & Modern CSS
